Masters Project

Bloch Sphere Visualisations with QuTIP

Website

My Masters work at UCL involves Matlab, Mathematica and Python simulations of mean-field and fully quantum descriptions of the Jaynes-Cummings system in the dispersive regime, with particular interest in the development of bistabilities in the system quasi-probability representations.

QuTIP is a FOSS python framework for modelling open quantum systems with particular applications in quantum optics. The two-level system (and analagously multilevel systems, though less usefully) can be simply represented on the surface of a sphere by way of a particular isomorphism. QuTIP provides a class for visualisation of such system dynamics.

This website is built with Bootstrap in Sublime Text 3, using markdown. It runs on a DigitalOcean instance using DNSimple and Namecheap for DNS and registration. The blog is built with Jekyll. Most products are from the Student Developer Pack from GitHub
